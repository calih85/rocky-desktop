# 🪨 Project Rocky

> "Friend."

Project Rocky is an AI desktop companion inspired by the spirit of *Project Hail Mary*.

The goal is to create an AI companion that lives on the desktop and assists users with productivity, learning, and coding while providing a fun and friendly experience.

---

## ✨ Planned Features

- 🪨 Desktop companion
- 💭 Thought bubbles
- 😊 Personality and emotions
- 🧠 AI conversations
- 💾 Memory
- 📅 Reminders
- 💻 Coding assistant
- 📚 Study companion
- 🎮 Fun interactions

---

## 🛠️ Tech Stack

- Electron
- JavaScript (later TypeScript)
- HTML
- CSS
- Node.js
- Git
- GitHub
